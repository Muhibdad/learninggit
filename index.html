There are two options:

  1. Github Desktop
  2. Or Command Line

To use Command Line Following are the important commands:


git init
git add README.md
git commit -m "first commit"
git branch -M master
git remote add origin https://github.com/Muhibdad/learninggit.git
git push -u origin master

You can push an existing Directory through following commands:


git remote add origin https://github.com/Muhibdad/learninggit.git
git branch -M master
git push -u origin master
